# Motion
An underwater speedruning co-op game made in Godot

## Installation
Windows
- Download a release from [Releases](https://github.com/ketr4x/Motion/releases)
- Unzip the archive
- Run the .exe

Web:
- Play [here](https://motion-nexus.netlify.app/)

## Features
- Online co-op
- Fun gameplay
- Leaderboards
- Speedruning
